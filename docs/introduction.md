# Introduction

Darion Docs is the technical documentation home for Darion Technologies. It is
designed for engineers, architects, and operators who need precise information
without visual noise.

## Purpose

The documentation platform provides three durable layers:

| Layer | Purpose |
| --- | --- |
| Fundamentals | Explain the concepts and architecture that remain stable over time. |
| Guides | Describe task-oriented workflows for local development and integration. |
| Reference | Capture schemas, parameters, responses, and compatibility rules. |

## Writing Model

Each article should start with the reader's goal, then move into the minimum
background needed to complete that goal. Use reference tables for stable facts,
and use numbered procedures only when the order of operations matters.
